# Movie App

A React application that allows users to search and browse movies using the TMDB API.

## Features

- Search for movies
- View movie details
- Add movies to favorites
- Responsive design
- Real-time search

## Setup

1. Clone the repository:
```bash
git clone https://github.com/Bkumar-007/Movie-Rendering-React.git
cd Movie-Rendering-React
```

2. Install dependencies:
```bash
npm install
```

3. Create a `.env` file in the root directory and add your TMDB API key:
```
VITE_TMDB_API_KEY=your_api_key_here
```

4. Start the development server:
```bash
npm run dev
```

## Environment Variables

- `VITE_TMDB_API_KEY`: Your TMDB API key (required)

## Technologies Used

- React
- Vite
- TMDB API
- CSS
- React Router
- Context API
